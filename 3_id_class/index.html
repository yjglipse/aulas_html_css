ID E CLASS

CLASS

EXISTEM DUAS CARACTERISTICAS PRINCIPAIS DO CLASS

  • É possivel utilizar o mesmo class em varios elementeos diferentes
  • É possivel utilizar varias classes no MESMO elemento

ID

Existem duas CARACTERISTICAS principais do ID que são contrastantes com as caracteristicas do class

  • Não é possivel utilizar o mesmo ID em varios elementos diferentes
  • Não é possivel utilizar VARIOS IDS no mesmo elemento

O ID funciona como um CPF,NÃO pode existir dois iguais e a mesma pessoa(elemento) não pode ter dois cpfs
